## ðŸ“ Common Markdown Syntax

### Headings
```markdown
# Large Heading (H1)
## Medium Heading (H2)
### Small Heading (H3)
```

### Text Formatting
```markdown
**Bold text**
*Italic text*
***Bold and italic***
~~Strikethrough~~
`Inline code`
```

### Lists
```markdown
- Bullet point 1
- Bullet point 2
  - Nested item
  
1. First item
2. Second item
3. Third item

- [ ] Unchecked task
- [x] Checked task
```

### Links & Images
```markdown
[Link text](https://example.com)
![Image alt text](image-url.png)
```

### Quotes & Code
```markdown
> This is a quote
> Multi-line quote

`inline code` in a sentence

```javascript
// Code block with syntax highlighting
function hello() {
  console.log('Hello!');
}
```
```

### Tables
```markdown
| Header 1 | Header 2 | Header 3 |
|----------|----------|----------|
| Cell 1   | Cell 2   | Cell 3   |
| Data A   | Data B   | Data C   |
```

### Other
```markdown

## ðŸ’¡ Pro Tips

### 1. **Live Preview is Your Friend**
Type on the left, watch it render on the right. Experiment!

### 2. **Save Frequently**
Press `Ctrl+S` often. The editor doesn't auto-save (yet).

### 3. **Use Metrics**
Instead of: "Improved performance"
Write: "Improved performance by 45%, reducing load time from 3s to 1.5s"

### 4. **Be Specific**
Instead of: "Used various technologies"
Write: "**Technologies:** React, TypeScript, Node.js, PostgreSQL, Docker, AWS"

### 5. **Paste Images**
Copy any image (screenshot, logo, diagram) and paste with `Ctrl+V`

### 6. **Use the Toolbar**
Don't remember markdown syntax? Just use the toolbar buttons!

### 7. **Keep It Organized**
Use consistent heading levels:
- `#` for your name (H1)
- `##` for main sections (H2)
- `###` for subsections (H3)

## ðŸš¨ Common Issues

### Q: My changes aren't saving!
**A:** Make sure to click the **Save** button (top right) or press `Ctrl+S`

### Q: Preview not updating?
**A:** Check that you're typing in the left pane (editor), not trying to edit the right pane (preview)

### Q: Formatting looks weird
**A:** Make sure you have proper markdown syntax (spaces after `#`, `*`, `-`, etc.)

### Q: Can't paste images
**A:** Make sure your cursor is in the editor (left pane) when pasting

### Q: Toolbar disappeared
**A:** Scroll up - toolbar is at the top of the editor
