**DO:**

**DON'T:**
- Mix more than 3 bright colors in one section
- Use bright colors for large text blocks (readability)
- Use yellow or pink on white backgrounds without checking contrast
- Overuse gradients - reserve for special moments

## Typography

### Font Family

**Primary Font:** Inter
- Modern, clean, highly readable
- Excellent web performance
- Wide range of weights

### Type Scale

```
Heading 1: 48px (3rem) - Bold (700)
Heading 2: 36px (2.25rem) - Bold (700)
Heading 3: 30px (1.875rem) - Semibold (600)
Heading 4: 24px (1.5rem) - Semibold (600)
Heading 5: 20px (1.25rem) - Semibold (600)
Body Large: 18px (1.125rem) - Regular (400)
Body: 16px (1rem) - Regular (400)
Body Small: 14px (0.875rem) - Regular (400)
Caption: 12px (0.75rem) - Regular (400)
```

### Typography Guidelines

**Headlines:**
- Use bold weights (600-700) for impact
- Add generous line-height (1.2-1.4) for readability
- Consider using gradients on hero headlines
- Keep headlines punchy and energetic

**Body Text:**
- Regular weight (400) for readability
- Line-height: 1.6-1.8 for comfortable reading
- Maximum line length: 70 characters
- Use muted colors for secondary text

**DO:**
- Use semibold (600) to emphasize important words
- Break up long text with subheadings
- Add emoji to headlines for personality (sparingly)
- Increase letter-spacing slightly for all-caps text

**DON'T:**
- Use light weights on small text
- Center-align long paragraphs
- Use all-caps for body text
- Stack more than 3 heading levels visually

## Voice & Tone

### Brand Voice

Our voice is consistent across all touchpoints: **celebratory, supportive, and energetic**.

#### Characteristics

**Celebratory**
- Recognize every achievement, big or small
- Use enthusiastic language
- Example: "That's awesome!" vs "Acknowledged"

**Supportive**
- Act as a cheerleader, not a critic
- Offer encouragement
- Example: "You've got this!" vs "Proceed"

**Energetic**
- Use active voice
- Keep sentences punchy
- Example: "Let's build your portfolio!" vs "Portfolio creation is available"

**Friendly**
- Write like talking to a friend
- Use contractions
- Example: "We'll help you shine" vs "We will assist you"

**Optimistic**
- Focus on possibilities
- Frame positively
- Example: "Your next great role awaits!" vs "Continue job searching"

### Tone Variations

Our tone adapts to context while maintaining the voice:

#### Celebratory Moments (High Energy)
- Tone: Excited, emphatic
- Use: Success messages, achievements, milestones
- Example: "üéâ Resume generated! It looks amazing!"

#### Guidance (Medium Energy)
- Tone: Encouraging, helpful
- Use: Instructions, tips, feature explanations
- Example: "Add your work experience and watch your portfolio come to life!"

#### Errors (Calm Energy)
- Tone: Supportive, solution-focused
- Use: Error messages, blockers
- Example: "Oops! That didn't work. Let's try again together."

#### Professional Context (Controlled Energy)
- Tone: Warm professional
- Use: Resumes, cover letters, public portfolios
- Example: "Experienced software engineer with a passion for building impactful products"

### Copy Guidelines

**DO:**
- Start with verbs (action-oriented)
- Use emojis in UI (sparingly - 1-2 per screen)
- Celebrate user wins
- Keep it conversational
- Use "you" and "your" (user-focused)

**DON'T:**
- Use corporate jargon
- Be overly formal
- Apologize excessively
- Use negative language
- Write long paragraphs

## Component Patterns

### Buttons

#### Primary Button
- **Color:** Applause Purple background, white text
- **Hover:** Slight opacity change or scale (1.02)
- **Active:** Confetti animation (on success actions)
- **Text:** Action verbs ("Start Now", "Create Portfolio", "Generate Resume")

#### Secondary Button
- **Color:** White background, purple border, purple text
- **Hover:** Light purple background
- **Text:** Supporting actions ("Learn More", "Preview")

#### Success Button
- **Color:** Success Green background, white text
- **Usage:** Completion actions, positive confirmations

### Cards

#### Standard Card
- **Background:** White or cream
- **Border:** Light gray, 1px
- **Border Radius:** 12px (rounded-lg)
- **Shadow:** Subtle, increases on hover
- **Hover:** Lift effect (translateY: -2px)

#### Feature Card
- **Icon:** Colored background circle
- **Hover:** Border color changes to brand color
- **Animation:** Smooth scale on hover

#### Gradient Card
- **Background:** Purple to pink gradient
- **Text:** White
- **Usage:** Premium features, special call-outs

### Animations

#### Micro-animations
- **Hover:** Scale (1.02-1.05), translateY, opacity
- **Click:** Scale down (0.98) then release
- **Duration:** 150-250ms
- **Easing:** ease-out

#### Celebration Animations
- **Confetti:** On major achievements
- **Bounce:** On button clicks
- **Fade Up:** On page load
- **Stagger:** Sequential animations

#### Loading States
- **Style:** Playful spinner or pulsing elements
- **Color:** Brand colors
- **Text:** Encouraging ("Almost there!", "Working our magic!")

### Spacing Scale

```

md: 16px (1rem)
lg: 24px (1.5rem)
xl: 32px (2rem)
```

### Layout Principles

**Whitespace**
- Generous padding around elements
- Breathing room between sections
- Don't cram - less is more

**Hierarchy**
- Clear visual hierarchy with size, weight, color
- Guide eye to primary actions
- Group related elements

**Consistency**
- Consistent spacing between similar elements
- Align to grid when possible
- Maintain padding ratios

## Accessibility

### Color Contrast

All text must meet WCAG AA standards:
- **Normal text:** 4.5:1 contrast ratio minimum
- **Large text:** 3:1 contrast ratio minimum
- **Interactive elements:** 3:1 contrast ratio

#### Contrast Checks
- Applause Purple on white: ‚úÖ AA Pass
- Celebration Pink on white: ‚ö†Ô∏è Use for large text only
- Success Green on white: ‚úÖ AA Pass
- Text colors on backgrounds: All pass ‚úÖ

### Motion & Animation

**Respect `prefers-reduced-motion`:**
```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
```

- Use ring color (purple) at 2px width
- Never remove focus styles
